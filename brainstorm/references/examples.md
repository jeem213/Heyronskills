# Brainstorm Examples

## Example 1: AI Agent Business

**Challenge:** How to make money with AI agents?

### Divergence Approaches

| Approach | Description | Key Idea |
|----------|-------------|----------|
| A - Service | Sell AI agent development services | Hourly consulting |
| B - Product | Build reusable agent templates | SaaS subscriptions |
| C - Platform | Create agent marketplace | Commission fees |
| D - Content | Teach others to build agents | Courses + community |
| E - Hybrid | All of the above + automation | Full-stack business |

### Cross-Pollination
Combined A (service) + D (content) = Service + course upsell

### Amplification
Make it 10x: Build once, sell infinite copies via automation

## Example 2: Content Creation

**Challenge:** How to create daily content?

### Divergence Approaches

| Approach | Description |
|----------|-------------|
| A - Video | Daily video diary |
| B - Text | Written blog posts |
| C - Audio | Voice notes / podcast |
| D - Images | AI-generated art |
| E - Interactive | AI conversations live |

### Final Solution (Amplified)
Daily video diary with AI-generated images, transcribed to blog, clips for social media, voice note for podcast feed

---

## Use This Reference

When brainstorming, reference these patterns:
1. Divergence first - generate 5+
2. Cross-pollinate - don't just pick one
3. Amplify - make it bolder
4. Score - use the matrix
5. Save - commit to memory
