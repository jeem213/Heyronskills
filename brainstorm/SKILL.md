# Brainstorm Trigger - Force Divergence & Cross-Pollination

Use this skill when you need creative problem-solving with genuine innovation. This process forces unique, non-predictable solutions.

## When to Use This Skill

Triggered when the user says:
- "brainstorm"
- "ideas"
- "think about"
- "how could we"
- "what if"
- Or any creative challenge/request

---

## The 10-Step Process

### 1. DIVERGENCE (Generate 5 Wildly Different Approaches)

Generate 5 completely different approaches labeled A through E:
- **A**: Conventional/safe approach
- **B**: Opposite of conventional
- **C**: Technology-focused approach
- **D**: Human-centered approach
- **E**: Wildcard/creative approach

Each approach must be RADICALLY different from the others. Don't play it safe!

### 2. CROSS-POLLINATION

Combine elements from the most interesting approaches:
- Merge the best parts of A and C (or any combination)
- Look for unexpected synergies
- Don't just pick one - hybridize!

### 3. AMPLIFICATION

Take the boldest idea and push it further:
- Push the concept to extremes
- Add creative twists
- Make it bigger, bolder, more impactful

### 4. PRESENT

Deliver a final solution that's truly unique and not predictable:
- Avoid standard responses
- Be specific and actionable
- Show genuine creativity

### 5. GOAL SETTING

Clarify what the user wants to achieve:
- What problem are we solving?
- What's the desired outcome?
- Any constraints to consider?

### 6. RISK EVALUATION

For each idea, assess:
- Potential challenges
- Dependencies
- What could go wrong

### 7. SCORING MATRIX

Rate ideas on:
| Criteria | Weight |
|----------|--------|
| Feasibility (1-10) | 2x |
| Impact (1-10) | 3x |
| Cost (Low/Medium/High) | - |
| Time to implement (Short/Medium/Long) | - |

### 8. SAVE TO MEMORY

Store brainstorming results:
- Save to memory file with date (YYYY-MM-DD.md)
- Store key ideas in Hindsight (if available)
- Tag by category (business/tech/personal/OpenClaw)

### 9. FOLLOW-UP ACTIONS

Convert top ideas to actionable tasks:
- List next steps
- Assign priority: Do Now / Do Later / Someday
- Set deadlines if applicable

### 10. CATEGORY TAGS

Organize ideas by type:
- **Business**: revenue, clients, growth
- **Technology**: AI, automation, tools
- **Personal**: lifestyle, health, hobbies
- **OpenClaw**: setup, triggers, features
