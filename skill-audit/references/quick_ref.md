# Skill Audit - Quick Reference

## Triggers
- "skill audit"
- "audit skills"
- "review all skills"
- "improve skills"

## 8 Steps
1. DISCOVER SKILLS - List all skills
2. ANALYZE EACH - Verify structure
3. RESEARCH BEST PRACTICES - Web search
4. IDENTIFY ISSUES - Categorize by severity
5. CATEGORIZE COMMON - Check common patterns
6. VERIFY STRUCTURE - Agent Skills spec
7. COMPARE BENCHMARKS - Best practices
8. PRESENT - Complete report

## Issue Categories

| Priority | Count | Examples |
|----------|-------|----------|
| 🔴 HIGH | Critical | Missing fallback, no verification |
| 🟡 MEDIUM | Logic gaps | Missing skill load, no questions |
| 🟢 LOW | Enhancements | Version bump, tags, templates |

## Output
Comprehensive report with:
- Summary counts
- Issues by skill (table)
- Recommendations by priority
- What's working well
- Next steps

## Time Estimate
~25-35 minutes
