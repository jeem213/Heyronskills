# Heyron Skills 🦫

15 powerful trigger words to supercharge your AI agent.

## What Is This?

Heyron Skills is a library of ready-to-use trigger commands for OpenClaw AI agents. Each skill enhances your AI with specific capabilities - from deep research to creative brainstorming to memory management.

## What's Included

| Category | Skills |
|----------|--------|
| **Research** | analyze, upgrades |
| **Creativity** | brainstorm, creative-spark |
| **Memory** | study, remember, full-sync, hindsight |
| **Mega** | mega-brain, mega-dive, mega-sync |
| **Personal** | diary, update-sara |
| **Utility** | skill-audit, advise |

## Quick Start

1. **Clone this repo** or download individual skills
2. **Choose a skill** that fits your needs
3. **Copy the SKILL.md contents** into your OpenClaw configuration
4. **Use the trigger word** to activate!

## Skill Overview

### 🔬 Research Skills
- **analyze** - Deep research with sources, confidence levels, and step-by-step reasoning
- **upgrades** - Research AI improvements, check versions, get recommendations

### 💡 Creativity Skills  
- **brainstorm** - Force divergence & cross-pollination for wild ideas
- **creative-spark** - Generate creative approaches + motivational boost

### 🧠 Memory Skills
- **study** - Refresh memory, read context files, query Hindsight
- **remember** - Save current session to memory + GitHub backup
- **full-sync** - Check all systems (OpenClaw, Hindsight, GitHub)
- **hindsight** - 18-point memory health check

### 🚀 Mega Skills (Combos)
- **mega-brain** - Become instant expert in any topic
- **mega-dive** - Research + creative solutions + tech upgrades
- **mega-sync** - Full sync + Hindsight + GitHub in one

### 💜 Personal Skills
- **diary** - Daily journaling with AI reflection
- **update-sara** - Morning update for your partner with weather, quotes, horoscope

### 🔧 Utility Skills
- **skill-audit** - Review and improve all trigger documentation
- **advise** - Get recommendations for any topic

## Example Usage

```
You: "analyze: What are the best AI coding models in 2026?"

Stuart: *researches thoroughly, shows sources, indicates confidence level*
```
