## 8-Step ReAct Process
1. THOUGHT - Identify question, knowledge gaps
2. ACT - Use tools (web_search, web_fetch, memory_search, image, pdf)
3. OBSERVATION - Incorporate findings, note contradictions
4. RESPOND - Show reasoning + confidence level
5. SOURCE CITATION - MUST cite ALL sources
6. CONFIDENCE CALIBRATION - 🔴🟠🟡🟢 Very Low to Very High
7. SELF-VERIFICATION - Double-check facts
8. MULTI-SOURCE VERIFICATION - 2+ sources, Hindsight, KB

## Output
- Transparent reasoning
- Confidence level
- Clear answer
- ALL sources cited
